Working With Strings By Chloe

Before beginning to learn about strings it is important to understand the difference between string values and string literals:
A string literal is what you see in the source code of a computer program, including the quotation marks.
A string value is what you see when the print() function is called, without the quotations marks.

String Concatenation

This is the joining of strings.
To concatenate two strings together, you use the '+' operator.
If a whitespace is needed, include a whitespace in one of the strings. 
        first_name = "Chloe "
        last_name = "Ridley"
        full_name = first_name + last_name
        print("full_name")
        =>
        "Chloe Ridley"
If no whitespace was included in the first_name variable, the result would have been ChloeRidley because Python would not have known to include a whitespace.
When concatenating two or more strings together, Python is creating a new string that can be used throughout the programme.

String Replication

In python you can repeat a string multiple times using the * operater:
        name = "Chloe"
        print(name * 3)
        =>
        ChloeChloeChloe
As with concatenation, if you want spaces in the output, they need to be included in the original string.

Formatting Strings

Sometimes additional formatting is needed within a string to ensure string values are displayed as intended for the user.
To span a string over multi-lines you can use either a triple single quotation or triple double quotations marks. 
        '''
        I am
        a multi
        line
        string.
        '''
        
        """
        I am also
        a multi
        line
        string.
        """
Escape characters can also be used to help format strings, some of the most common are:
NOTATION WHAT IT DOES
\ Tells Python to treat the quotation following it as part of the string
\n Starts a new line in the string.
\t Indents the line in the string.

Escape characters work well for short strings, but for longer strings the multi-line technique is preferred.
Sometimes the quotation marks or backslashes are needed in the string literal, this is where raw formatting can be used. This tell python to ignore all the formatting within a string and include all escape characters.

        str = r"hello\how\'are\you\m'k"

        print(str) => hello\how\'are\you\m'k
