Python Math Operators

The most commonly used math opertors are:
OPERATOR WHAT IT DOES
x + y Addition
x - y Subtraction
x * y Multiplication
x / y Division
x % y Modulo. Returns remainder of x divided by y
x // y Floor division. Returns an int
-x Negates x
x** To the power of

Python also uses compound operators:
OPERATOR WHAT IT DOES
x += Adds then assigns
x -= Subtracts then assigns
x *= Multiplies then assigns
x /= Divides then assigns
x %= Modulo then assign
x //= Floor division then assign
x**= To the power of then assign

Built-in Math Functions

Python has a number of built in math functions, to help speed up programming and help the developer to write less code.

abs() - This finds the absolute values, it does not take into account any negative values.
i.e abs(-19) => 19

This is useful for problems including distance travelled etc.
E.g. If you needed to travel 100km to a destination, but have accidently travelled 150km, if you calculated how many km you have to travel to the destination, it would give a negative number. This is where abs() can be used to help! 
        km_to_dest = 100
        km_travelled = 150
        km-left = km_to_dest - km-travelled
        print(km-left)
        =>
        -50
        print(abs(km-left))
        =>
        50
You are most likely to use abs() in Python when there is a possibility of result being negative or postive but you only want a postive value.

divmod() - This function takes two values, which can be either int or floats. It finds the floor quotient and remainder in one function.
i.e divmod(a, b) = a//b and a%b combined into one function. 
        divmod(20, 6)
        =>
        (3, 2)
pow() - provides an alternative to the ** syntax and allows more arguments to be passed. As well as variables to be passed into the argument, whereas you can not use the ** with variable names 
        num = 7
        print(pow(num, 2))
        =>
        49
round() - This method takes two arguments. The first is the number to be rounded and the second is to how many decimal places. 
        i = 18.14683280273
        print(round(i, 2))
        =>
        18.15
sum() - This method is used to find the sum of numeric compound data types, including lists, tuples and dictionaries. It can take up to two arguments 
        i = [1, 2, 3]
        print(sum(i, 5))
        =>
        11
