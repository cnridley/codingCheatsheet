Converting Data Types By Chloe

Different data types are sometimes converted into other data types in order to manipulate values in different ways.
For example, if you tried to add a string with a number you would receive an error, therefore you need to convert either the string to a number(if possible) or the number to a string before concatenating them.
You also can't modify a tuple, so therefore would have to convert it to a list beforehand and then modify it accordingly.

Converting Number Types

To convert from int to float:
Use the float() method. 
        my_int = 67
        print(float(my_int))
        =>
        67.0
To convert from float to int:
Use the int() method.
        my_float = 6219.792
        print(int(my_float))
        =>
        6219
When converting from a float to an int, Python does not do any rounding, it simply just cuts off the decimal point.

Converting Numbers to Strings

Use the str() method. This is useful because in Python you can not concatenate numbers and strings together, it would return an error. i.e 
        name = "chloe"
        age = 27
        birthday = "Happy Birthday" + name + " you are " + age + "today!"
That would return an error, instead you would have to convert the age to a string: 
        birthday = "Happy Birthday" + name + " you are " + str(age) + "today!"
You would use the exact same method for floats!

Converting Strings to Numbers

Strings can be converted to numbers using the int() and float() method as outlined above.
This is useful when we need to apply mathematical operations on numerical data that has been inputted as a string. Using the + operator has a differnt output when used on two strings: 
        num1 = "5"
        num2 = "6"
        print(num1 + num2)
        =>
        56
This is probably not the output the user wanted, which is why we convert to a number type first: 
        print(int(num1) + int(num2))
        =>
        11
You can also not use the '-' operator on a string as it would return an error, which is another example of when you would need to convert to a number first.

Converting to a tuple

Use the tuple() method.
Converting to a tuple can allow substantial optimization to program we create. It also tells any other developers that you have no intention of changing the values of the items in the tuple. 
        my_list = ['giant', 'evil', 'liv']
        print(tuple(my_list)
        =>
        ('giant', 'evil', 'liv')
Any iterable data type can be converted to a tuple (int and floats cannot be converted) 
        name = "Chloe"
        print(tuple(name))
        =>
        ('C','h','l','o','e')

Converting To Lists

Use the list() method.
Useful when you need a mutable data type.
For example, when you have a tuple which needs to be modified. 
        my_tuple = ('giant', 'evil', 'liv')
        print(list(my_tuple))
        =>
        ['giant', 'evil', 'liv']
This can then be modified and then changed back into a tuple if needed.
Any iterable data type can be converted to a tuple (int and floats cannot be converted) 
        name = "Chloe"
        print(list(name))
        =>
        ['C','h','l','o','e']
