Python Tuples By Chloe

Python tuples are an immutable data type, meaning they cannot be changed. When you use a tuple you are conveying to others that you don't intend any changes to the data.

Tuples are defined using the ()
The following is an example of a tuple, which will be used throughout this page. 
        my_tuple = ('liv', 'norco', 'felt')
If you needed to modify a tuple, you would have to convert to a list first.

Indexing Tuples

Tuples are indexed in the same way as lists. Indexing starts at 0 i.e 'liv' = 0.
However, with a long tuple you can index it negatively to access items towards the end of the list, when doing this it always starts with -1. i.e. 'felt' = -1.

To access items using index numbers you use the []. 
        print(my_tuple[1])
        =>
        norco
        print(my_tuple[-3])
        =>
        liv

Concatenating Tuples

You can concatenate tuples two different ways:
The first is concatenating a string with a tuple item using the + operator: 
        print("Chloe rides a " + my_tuple[0])
        =>
        Chloe rides a liv
The second way is using the + operator to concatenate two tuples together to make a new tuple. 
        my_tuple = ('liv', 'norco', 'felt')
        brands = ('fox', 'poc', 'dakine')

        print(my_tuple + brands)
        =>
        ('liv', 'norco', 'felt', 'fox', 'poc', 'dakine')

Slicing and Striding Tuples

Slicing allows us to call multiple items using the [x:y] syntax, where x is inclusive and y is exclusive. 
        tuple = ('liv', 'norco', 'felt', 'fox', 'poc', 'dakine')
        print(tuple[2:4])
        =>
        ('felt', 'fox')
If you only wanted to print one end of the tuple, omit either the x or y from the syntax. 
        tuple = ('liv', 'norco', 'felt', 'fox', 'poc', 'dakine')
        print(tuple[:4])
        =>
        ('liv', 'norco', 'felt', 'fox')

        print(tuple[3:])
        =>
        ('fox', 'poc', 'dakine')
The stride is the optional third parameter, it defines how many items to move forward each time. The default is 1. The syntax for this is [x:y:z] 
        tuple = ('liv', 'norco', 'felt', 'fox', 'poc', 'dakine')
        print(tuple[1:5:2])
        =>
        ('norco', 'fox')
If you just wanted to skip an item each time you could do the following: 
        tuple = ('liv', 'norco', 'felt', 'fox', 'poc', 'dakine')
        print(tuple[::2])
        =>
        ('liv', 'felt', 'poc')

Tuple Function

Python has built in functions to tell us information about the Tuple:
FUNCTION WHAT IT DOES
len() Tells us how many items in a Tuple.
max() Find highest value in Tuple.
min() Finds lowest value in Tuple.

The min and max function are useful to help find information about quantitative data. E.g. test scores, temperature or prices.

As well as these functions you can also multiply tuples together using the * operator. This is useful if you needed two copies of all files in the directory. 
        print(my_tuple*2)
        =>
        ('liv', 'norco', 'felt', 'liv', 'norco', 'felt')
