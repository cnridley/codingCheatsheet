Python Dictionaries By Chloe

Dictionaries are Pythons built-in mapping type. They are used to hold data that are related to each other. Dictionaries are defined using the {}, and made up of key-value pairs which are separated by :

Below is an example of a Python Dictionary, we will use this example throughout this page: 
        profile = {'name':'Chloe', 'gender':'female', 'age': 27, 'citizen':'UK'}

Accessing and Modifying Python Dictionaries

To access values items in a dictionary you can use the []. 
    print(profile['name])
    =>
    Chloe
To add a key-value pair to a dictionary, you use the syntax dict_name['key'] = value. As Dictionaries are unordered, when added they can appear anywhere in the dictionary. 
   profile['hobby'] = 'biking'
   print(profile)
   =>
   {'name': 'Chloe', 'gender': 'female', 'age': 27, 'citizen': 'UK', 'hobby': 'biking'}
Similarly, to update existing key value pairs use the syntax dict_name['existing-key'] = new value. 
        profile['name'] = 'George'
        print(profile)
        =>
        {'name': 'George', 'gender': 'female', 'age': 27, 'citizen': 'UK'}
When deleting from a dictionary, there are a couple of methods you can do:

del dict_name[key] - This will delete the key-value pair stated in [].
dict_name.clear() - This will clear the entire dictionary but keep the empty dictionary.
del dict_name - This will delete the entire dictionary.

Dictionary Function

FUNCTION WHAT IT DOES
dict_name.keys() Returns all the keys in list format
dict_name.values() Returns all the values in list format
dict_name.items() Returns all the values in a list format 

        profile.keys()
        =>
        dict_keys(['name', 'gender', 'age', 'citizen'])

        profile.values()
        =>
        dict_values(['Chloe', 'female', 27, 'UK'])

        profile.items()
        =>
        dict_items([('name', 'Chloe'), ('gender', 'female'), ('age', 27), ('citizen', 'UK')])
You can also use a for loop to make list more user friendly and readable.
NOTE that if you have a number data type as one of the values, you have to use the str() method. 
        profile = {'name':'Chloe', 'gender':'female', 'age': 27, 'citizen':'UK'}

        for key, value in profile.items():
        print(key +' is the key for the value ' + str(value))

        =>
        name is the key for the value Chloe
        gender is the key for the value female
        age is the key for the value 27
        citizen is the key for the value UK
