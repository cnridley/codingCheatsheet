Python Lists By Chloe

Lists are useful when you want to work with many related values as it enables you to keep data together, condense your code an perform the same methods and operations on multiple values at once.
They are a mutable data type and are defined using the [].

The following list will be used for all the examples on this page: 
        my_list = ['bike', 'hike', 'ski', 'run', 'swim']

List Indexing

Every item in a list has its own index number. Indexing a list always starts at 0.
i.e. 'bike' would have an index of 0 and 'run' would have an index of 3.

When you have a long list and need to access an item towards the end of the list, you can index your list from negative numbers. When using negative numbers, indexing always starts with -1.
i.e. 'swim' would be -1 and 'bike' would be -5.

We are able to access and manipulate our lists through the index numbers. 
        print(my_list[2])
        =>
        ski

        print(my_list[-2])
        =>
        run
Using list indexing you are able to concatenate with strings using the + operator. 
        print("Chloe likes to " + my_list[0] +".")
        =>
        Chloe likes to bike.
You can also modify list items by using the index numbers and then assigning it to a new value using the = operator. 
        my_list[2] = "Walk"
        print(my_list)
        =>
        ['bike', 'hike', 'Walk', 'run', 'swim']
As you can see 'ski' has been replaced with 'walk' because we assigned it a new value, this allows us to change and update lists in an effcient way.

Slicing a list means you can call multiple list items using the [x:y] syntax.
The x is inclusive and the y is exclusive to the result. 
        print(my_list[1:4])
        =>
        ['hike', 'ski', 'run']
Using the index numbers you can also delete items from the list using the del keyword. You can either delete one item or a range of items using the [x:y] syntax, x is inclusive, y is exclusive. 
        del my_list[2]
        print(my_list)
        =>
        ['bike', 'hike', 'run', 'swim']

        del my_list[1:4]
        print(my_list)
        =>
        ['bike', 'swim']

List Methods

Something you should note before learning list methods is that when applying a list method to a list you are affecting a list itself, not a copy. It is useful to make a copy of the orginal list just in case!

METHOD WHAT IT DOES
list.append(x) Will add 'x' as an item to the end of the list.
list.insert(i, x) Add 'x' to index position 'i'
list.extend(L) L is a second list
list.remove(x) Removes 'x'. If multiple x's, the first is removed
list.pop([i]) Return the item at i and then remove item.
list.index(x) Returns index number of x. If multiple x's, returns first one.
list.copy() Copies list
list.reverse() Reverses order of list.
list.count(x) Returns the number of times x occurs
list.sort() Sorts items in a list. Useful to sort numbers
list.clear() Removes all items in a list.

As well as modifying lists using methods, you can also modifying lists using operators:
Using the + operator you can either concatenate two list together or add a list item to the end of the list.
        print(my_list + ['walk'])
        =>
        ['bike', 'hike', 'ski', 'run', 'swim', 'walk']

        my_list = ['bike', 'hike', 'ski', 'run', 'swim']

        my_second_list = ['coffee', 'tea', 'water']

        print(my_list + my_second_list)
        =>
        ['bike', 'hike', 'ski', 'run', 'swim', 'coffee', 'tea', 'water']
Using the * operator, you can multiply lists: 
        print(my_list * 3)
        =>
        ['bike', 'hike', 'ski', 'run', 'swim', 'bike', 'hike', 'ski', 'run', 'swim', 'bike', 'hike', 'ski', 'run', 'swim']

Nested Lists

A nested list is a list within a list.

my_nested_list = [['chloe', 'nicole', 'grace], ['George', 'Andrew', 'Ty']]

To access items within a nested list you need two indices.
my_nested_list[0][1] => nicole
my_nested_list[1][0] => George

List Comprehensions

List comprehensions offer a way to create lists based on existing lists. They provide an alternative syntax to creating lists and other sequential data types.

They are constructed as:
list_variable = [x for x in iterable]

    chloe_letters = [letter for letter in 'Chloe']
    print(chloe_letters)
    =>
    ['C', 'h', 'l', 'o', 'e']
chloe_letters is now a new list created using a list comprehension, with each letter being a new item.
Within list comprehensions, it is only the 'for' and 'in' that are keywords. I.e the following would give the same result. 
    chloe_letters = [cupOfTea for cupOfTea in 'Chloe']
    print(chloe_letters)
    =>
    ['C', 'h', 'l', 'o', 'e']
Not that you would ever use 'cupOfTea' but it is just an example!

The above example is using a string data type to create a new list, you can also do the same with tuples and lists: 
        tuple_example = [item for item in ('hello', 'bonjour', 'hola')]
        print(tuple_example)
        =>
        ['hello', 'bonjour', 'hola']

        list_example = [item for item in ['bye', 'cya', 'adios']]
        print(list_example)
        =>
        ['bye', 'cya', 'adios']
List comprehensions can be re-written as for loops but not every for loop can be written as a list comprhension. We will write the chloe_letter example as a for loop. 
        chloe_letters = []

        for letter in 'chloe':
        chloe_letters.append(letter)
        print(chloe_letters)
        =>
        ['c', 'h', 'l', 'o', 'e']
As with before, in a for loop it is only the 'for' and 'in' that are keywords. The following would give the same result: 
        chloe_letters = []

        for cupOfTea in 'chloe':
        chloe_letters.append(cupOfTea)
        print(chloe_letters)
        =>
        ['c', 'h', 'l', 'o', 'e']

Conditionals With List Comprehensions

Conditional statements can be used to control which items in an existings sequence are included in the new list. Below are two examples with an explanation of what is happening: 
        chloe_letters = [letter for letter in 'Chloe' if letter != 'e']
        print(chloe_letters)
        =>
        ['C', 'h', 'l', 'o']
In this example, we have a list comprehension which will create a new list, which each item will be a letter in the string 'chloe', but then have added an if statement to say only add to the new list if the item does not equal 'e'. Hence why 'e' has been left of the end. 
        numbers_list = [y*3 for y in range(20) if y%3 == 0]
        print(numbers_list)
        =>
        [0, 9, 18, 27, 36, 45, 54]
In this example Python would have gone through the following steps:
'y in range(20)' - Would have returned numbers 0-19
'y%3 == 0' - Would have checked if numbers 0-19 returned 0 when divided by 3.
0, 3, 6, 9, 12, 15, 18
'y*3' - Would have multiplied the above results to the power of 3 to get the final output.

Nested Loops In List Comprehensions

A nested list comprehension would look like: 
        my_list = [x*y for x in [20, 40, 60] for y in [1, 2, 3]]
        print(my_list)
        =>
        [20, 40, 60, 40, 80, 120, 60, 120, 180]
As a for loop that would be written as the following: 
        my_list = []
        for x in [20, 40, 60]:
            for y in [1, 2, 3]:
            my_list.append(x*y)
        print(my_list)
        =>
        [20, 40, 60, 40, 80, 120, 60, 120, 180]
You should note that although list comprehensions flatten the code ino one line, it is important to remember that code readibility is more important, therefore loops are sometimes preferred so that it is more user friendly.
