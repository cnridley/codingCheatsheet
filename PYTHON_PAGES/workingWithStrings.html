Working With Strings By Chloe

Before beginning to learn about strings it is important to understand the difference between string values and string literals:
A string literal is what you see in the source code of a computer program, including the quotation marks.
A string value is what you see when the print() function is called, without the quotations marks.

String Concatenation

This is the joining of strings.
To concatenate two strings together, you use the '+' operator.
If a whitespace is needed, include a whitespace in one of the strings. 
        first_name = "Chloe "
        last_name = "Ridley"
        full_name = first_name + last_name
        print("full_name")
        =>
        "Chloe Ridley"
If no whitespace was included in the first_name variable, the result would have been ChloeRidley because Python would not have known to include a whitespace.
When concatenating two or more strings together, Python is creating a new string that can be used throughout the programme.

String Replication

In python you can repeat a string multiple times using the * operater:
        name = "Chloe"
        print(name * 3)
        =>
        ChloeChloeChloe
As with concatenation, if you want spaces in the output, they need to be included in the original string.

Formatting Strings

Sometimes additional formatting is needed within a string to ensure string values are displayed as intended for the user.
To span a string over multi-lines you can use either a triple single quotation or triple double quotations marks. 
        '''
        I am
        a multi
        line
        string.
        '''
        
        """
        I am also
        a multi
        line
        string.
        """
Escape characters can also be used to help format strings, some of the most common are:
NOTATION WHAT IT DOES
\ Tells Python to treat the quotation following it as part of the string
\n Starts a new line in the string.
\t Indents the line in the string.

Escape characters work well for short strings, but for longer strings the multi-line technique is preferred.
Sometimes the quotation marks or backslashes are needed in the string literal, this is where raw formatting can be used. This tell python to ignore all the formatting within a string and include all escape characters.

        str = r"hello\how\'are\you\m'k"

        print(str) => hello\how\'are\you\m'k

String Functions

Built-in Python functions allow us to easily modify and manipulate strings.
You should note that strings are a immutable data type, meaning the returned string will be a new string, not the old one modified.

Two common functions are:
str.upper() - This converts all to upper case.
str.lower() - This converts all to lower case.
These two functions are useful when data is being inputed by a user which is then later evaluated by someone. Because Python is case sensitive, it allows all the data to be converted to lower or upper, so that other functions can be used on it and the evaluation is more accurate.

Python also has string functions that evaluate to a Boolean value:
FUNCTION WHAT IT DOES
str.isalnum() True is only alphanumeric values. No symbols
str.isalpha() True if only letters. No symbols
str.islower() True if letters are all lower case
str.isnumeric() True if consists of only numbers.
str.isspace() True if only whitespace characters
str.istitle() True if string is in title case
str.isupper() True if letters are all upper case.

It can be useful to check password parameters.
        my_length_string = "Hello my name is Chloe Ridley"

        print(len(my_length_string)) => 29

print(", ".join(['bike','hike','ski','swim']))

The str.split() method returns a list of strings that are seperated by whitespace if no parameters are given. It can also be used to remove parts of the string.

my_string = "hello my name is chloe"

The str.replace() method takes an original string and returns an updated string with some replacement. It takes two arguments, the first is the part of the string being replaced, the second is the replacement.

replace_string = "Good morning"
print(replace_string.replace("morning", "evening"))
=>
Good evening

The str.count() method counts the number of times a particular character or a sequence of characters show up in a string. It is case sensitive, which is where .upper and .lower methods come in handy.

The str.find() method returns the postion of a character based on index number. If there is more than one it will return the postion of the first one.

Indexing strings

Each character in a string corresponds to an index number, the first character being 0. Indexing strings allows us to manipulate and access strings within data.

Ie. In the string "hello World!", the 'h' would be index 0, the whitespace would be index 5 and the '!' would be 11.

Strings can also be indexed with negative numbers, this is useful when you need to access a character towards the end in a very long string. It starts at -1.

I.e in "hello World!", the '!' would be -1, the 'W' would be -6.

To access characters in a string using index numbers you used the square brackets.

my_string = "hello world!"
print(my_string[2])
=>
l

print(my_string[-6])
=>
w

Slicing strings.

You can take a 'slice' out of a string with the slice method.
It uses the syntax [x:y], where x is inclusive and y is exclusive.

my_string = "hello world!"
print(my_string[0:5])
=>
hello

If you only want to include either ends of the string you omit one of the numbers in the [x:y] syntax. i.e.

my_string = "hello world!"
print(my_string[:5])
=>
hello

my_string = "hello world!"
print(my_string[6:])
=>
world!

Stride in strings

Slicing strings can take a third parameter; the stride.
This is the number of characters Python moves forwards once the first character is received.
The default number is 1.

my_string = "hello world!"
print(my_string[0:12:2])
=>
hlowrd
This is because it skips every other character.

If you put;
my_string = "hello world!"
print(my_string[::-1])
=>
!dlrow olleh
It reverses the order of the string.

String Formatters

Formatters work by allowing you to put placeholders {} into a string and call the str.format() method later with a value to fill the placeholder.
        my_string = "Chloe enjoys {}"
        print(my_string.format("biking"))
        =>
        Chloe enjoys biking
You can put more than one argument in the parantheses of the format method. By default Python fills curly braces in the order of the arguments.
However, they are indexed in the same way that strings are, giving us more choices on how to modify the string.
        my_string = "Learning is {1}, especially {0}."
        print(my_string.format("fun", "Python"))
        =>
        Learning is fun, especially Python.
If it is a long placeholder value, you can asign it a keyword argument:
        my_string = "{0} the {1} developed a {in}."
        print(my_string.format("Chloe", "web developer", in="interactive website"))
        =>
        Chloe the web developer developed a interactive website.
When passing through a value in the {}, you can also use the {field_name:conversion} syntax.
The field_name tells Python the index number of the value in the str.format.
The conversion tells Python the conversion code of the data type that we are using within the formatter. To find out more about the conversion types visit the offical documents of Python, but some of the most common ones are:
s - string
d - decimal place (10-base)
f - floats with decimal place.
        my_money = "Chloe had {1:f} in her {0}"
        print(my_money.format("bank", 18.79))
        =>
        Chloe had 18.790000 in her bank.
Because we have used the 'f' conversion type, Python returns a float number, but as you can see there are too many decimal places. You can declare how many decimal place you want by doing the following. 
        print("Chloe had {1:.2f} in her {0}".format("bank", 18.79))
        =>
        Chloe had 18.79 in her bank.
By defining .2f in the placeholder, Python only puts two decimal places.
If 0 decimal places are wanted use .0f

Variables can also be passed through formatters. 
        cake = 2
        bday = "Chloe has {} cakes for her birthday!"
        print(bday.format(cake))
        =>
        Chloe has 2 cakes for her birthday!
With formatters you can also add padding to the placeholder by adding a number to the right of the :
> will align right
^ will align center
The less than symbol will align left. 
        my_string = "Chloe {0:>6} to {1:^16}"
        print(my_string.format("loves", "code"))
        =>
        Chloe  loves to       code
By default, Python will use whitespace unles otherwise stated. 
        my_string = "{:*^20s}"
        print(my_string.format("Chloe"))
        =>
        *******Chloe********
