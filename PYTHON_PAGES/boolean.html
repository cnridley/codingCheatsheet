Python Booleans By Chloe

Booleans are a data type that can only evaluate to True or False, always with a capital T and F. They are used to make comparsions and control the flow of a program.

Comparison Operators

Comparison operators can be used for int, floats and strings. They are used to compare values and evaluate to a Boolean value.
Common operators are:
OPERATOR WHAT IT MEANS
== Equal to
!= Not equal to
< Less than
> Greater than
<= Less than or equal to
>= Greater than or equal to

Python also has logical operators:
OPERATOR WHAT IT DOES
and True if both are true
or True if at least one is true
not True only if false

Truth Tables

Truth tables help to improve algorithmic thinking when programming:

== TRUTH TABLE:
X == Y RESULT
True == True True
True == False False
False == True False
False == False True

AND TRUTH TABLE:
X AND Y RESULT
True AND True True
True AND False False
False AND True False
False AND False FALSE

OR TRUTH TABLE:
X OR Y RESULT
True or True True
True or False True
False or True True
False or False False

NOT TRUTH TABLE:
NOT X RETURNS
Not True False
Not False True

Using Boolean operators to control flow

To control the flow of a program we can use a condition followed by a clause i.e.
If 'x' is true do this:

An example of this would be when grading someones paper. 
        grade = 81

        if grade >= 80:
        print("You passed!")
        else:
        print("Better luck next time!")
Because the grade = 81, the program would have printed "You Passed!"
